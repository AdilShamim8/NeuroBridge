# NeuroBridge v0.1.0 - Cognitive Accessibility for AI

AI has become a universal interface, but it still speaks in a single format that excludes millions of people with different cognitive needs. NeuroBridge exists to close that gap: same model, same answer intent, but adapted presentation that is easier to process for ADHD, autism, dyslexia, anxiety, and dyscalculia profiles.

## What's in v0.1.0

- Five built-in cognitive profiles with profile-specific transformation rules.
- Middleware-style integration for existing LLM clients.
- REST API server with profile, adapt, batch, stream, and quiz routes.
- CLI workflow (`info`, `adapt`, `quiz`, profile storage commands).
- Memory backends (SQLite and Redis fallback) with adaptive feedback loop.
- Streaming adaptation and format adapters (markdown/html/plain/json/tts).
- Security hardening: bearer API key auth, rate limits, request caps/timeouts, security headers.
- Privacy baseline: hashed feedback references and user export/delete capabilities.
- Demo and learning assets: HuggingFace Space app and Colab quickstart notebook.

## Quickstart

```python
from neurobridge import NeuroBridge, Profile

nb = NeuroBridge(llm_client=openai_client)
nb.set_profile(Profile.ADHD)
response = nb.chat("Explain how machine learning works")
print(response.adapted_text)
```
