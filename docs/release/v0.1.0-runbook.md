Date prepared: 2026-03-23

## 1. Local Verification

```bash
python -m pytest tests/ -q
python -m pytest --cov=neurobridge --cov-report=term --cov-fail-under=90 -q
python -m mypy neurobridge/
python -m ruff check neurobridge/
python -m black --check neurobridge/
python -m build
python -m twine check dist/*
python -m pip install --force-reinstall dist/neurobridge-0.1.0-py3-none-any.whl
python -c "from neurobridge import NeuroBridge, Profile"
neurobridge info
neurobridge adapt "test" --profile adhd --format plain
neurobridge serve --port 8099
```
