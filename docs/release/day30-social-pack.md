# Day 30 X/Twitter Launch Thread

## Tweet 1 - Hook

Most AI assistants still assume everyone processes language the same way.
They do not.

NeuroBridge is now live: open-source middleware that adapts AI output for ADHD, autism, dyslexia, anxiety, and dyscalculia.

## Tweet 2 - 2-line solution

2-line Python integration:

from neurobridge import NeuroBridge, Profile
bridge = NeuroBridge(); bridge.set_profile(Profile.ADHD)

Then adapt model output before sending it to users.

## Tweet 4 - Profiles

Profiles currently supported:
- ADHD: summary-first, reduced overload
- Autism: literal, explicit wording
- Dyslexia: shorter sentences and spacing
- Anxiety: calmer language, reduced urgency cues
- Dyscalculia: number context and simpler numeric framing
