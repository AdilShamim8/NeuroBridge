# Contributing

Contributions are welcome across docs, profiles, transforms, integrations, and accessibility QA.

## Development setup

```bash
git clone <repo>
cd NeuroBridge
pip install -e ".[dev]"
pytest -q
```

## Contribution checklist

- Add or update tests
- Update docs and changelog entries
- Keep profile behavior changes explicit and evidence-based
- Run lint and type checks before opening PR

## Reporting profile behavior issues

Use the profile improvement issue template and include concrete before/after examples.
