# Quiz

ProfileQuiz maps user preferences to a recommended adaptation profile.

## Python API

```python
from neurobridge import ProfileQuiz

quiz = ProfileQuiz()
result = quiz.evaluate([0, 1, 2, 3] * 4)
print(result.primary_profile)
```
